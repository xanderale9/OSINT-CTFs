# OSINT CTF Writeups

This repository contains writeups and solutions for various OSINT (Open Source Intelligence) Capture The Flag challenges from multiple organizations.

## Organizations Covered

- **[TraceLabs](./TraceLabs/)** - Missing persons OSINT investigations
- **[Hacktoria](./Hacktoria/)** - Web reconnaissance challenges
- **[OSINT UK](./OSINT_UK/)** - UK-based OSINT challenges
- **[Maltego](./Maltego/)** - Data visualization and intelligence gathering
- **[PicoCTF](./PicoCTF/)** - General CTF challenges with OSINT components

## How to Use This Repository

Each challenge directory contains:
- Detailed writeups of the challenge
- Tools and techniques used
- Solutions and findings
- Links to resources
