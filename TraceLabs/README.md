# TraceLabs OSINT CTF Writeups

TraceLabs is dedicated to improving OSINT skills through real-world missing persons investigations. These challenges involve gathering information from publicly available sources to help locate missing individuals.

## Challenge Structure

Each challenge typically includes:
- Case information about a missing person
- Specific objectives to complete
- Points awarded for correct information

## Tips for Success

1. **Use Multiple Sources**: Google, social media, maps, WHOIS, DNS lookups
2. **Document Everything**: Keep track of URLs and sources for your findings
3. **Cross-Reference**: Verify information through multiple sources
4. **Think Like an Investigator**: Look for patterns and connections
