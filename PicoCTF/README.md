# PicoCTF OSINT Challenges

PicoCTF is a beginner-friendly CTF platform that includes various challenge categories, including those with OSINT components.

## Challenge Categories Covered

- Web Exploitation with OSINT components
- Forensics challenges
- General reconnaissance

## Getting Started

1. Visit [picoctf.org](https://picoctf.org)
2. Create an account
3. Explore challenges by difficulty
4. Use writeups here to learn solutions and techniques
